Data Analysis and Visualizations

1. Intro

Data used for this analysis and visualization was sourced from the public dataset on Google BigQuery that was created by the Social Security Administration. This dataset contains all names from Social Security card applications for births that occurred in the United States after 1879. The data set does have limitations that should be noted, such as the fact that many people born before 1937 never applied for a Social Security card, so their names are not included in this data. For others who did apply, records may not show the place of birth, and again their names are not included in the data. Also, to safeguard privacy, the Social Security Administration restricts names to those with at least 5 occurrences.

2. Process

Data covers records from years 1910 to 2021. A copy was made and saved under my own BigQuery account in order to manipulate data for the visualizations. Two versions of the dataset were made, one is similar to the original file with a couple of columns added, and the other has the records grouped by decade. The original file has 6,311,504 individual records, while the decade version has 1,187,038. Original Columns used include:

  • State(two-char state code) - String
  • Gender(sex M = male/F = female) - String
  • Year(YYYY format ) - Integer
  • Name (given name of person at birth) - String
  • Number (Number of occurrences of name)

The original dataset on BigQuery Public Data has already been cleaned of null values. This was double-checked using a query, which didn't yield any null records. At this point there was a decision to create two datasets, one working with they years given originaly, and a second one that groups records into decades instead of years. Both records will have additional data added that would make it too complicated to have as one large table. Also, a third dataset was created that would be used for tracking certain details of each unique name. this thrid dataset, along with the extra columns on the previous two datasets where created with SQL queries on Google BiqQuery. More information on this can be found in the "Notes" Section of this site.

3. Analysis

3.1 Top 10 Baby Names

This Dashboard consolidates the top 10 Male and Female names in a given decade. The results (a word cloud) show names with the text size correponding to it's popularity. Also, hovering the mouse over the names will give additional information, such as their ranking during the decade at a national level and state level if a state is selected. This makes it easier to visualize how naming trends have changed in the last 100 years. It makes it clear how certain names went from being extremely popular (e.g. "Mary" being the top Female name for 40+ years) to Droping off completely from the top 10 rankin in one decade. It's also easier to see why certain names invoke a specific time period or perceptiong of age because of when they were most popular.

3.2 Name Popularity by Years

This second dashboard is similar to the first, but it focuses on individual names and their popularity over the years. Entering a name in the text box and selecting it will add shapes to the graph maping the number of people born with said name by year. The shapes are also divided by gender (via color) since some names have been used for both males and females. This last feature is useful for seeing how a name's popularity changes between men and women. Like the Previous dashboard, the tooltip also shows additional info upon hovering the mouse on a data point on the graph as well as filtering by state. You can also compare two or more names and see how their popularity has changed over time.

3.3 Name Details and Stats Pt. 1

This last dashboard is divided into two parts. This first part shows various graphs that indicate how male and female names vary from eachother in terms of length, count and composition of letters. One of the clearest difference is the variety of male and female names. Other studies on names have reached similar conclusions and have noted that women tend to have less ridgid societal standards to their names compaired to men. Also, the tendency for men to be named after relatives (e.g their father) can explain why available names remain more static. Females names have always had higher variety, but it seems to have really taken off during the post-war baby boom. More info on this can be found in notes section. Also, female names tend to be slightly longer with a average length of 6.35 to males' 6.06. Letter composition also varies, as can be seen by higher average vowel count in female names, while male names have a very slight higher consonant count.

3.4 Name Details and Stats Pt. 2

This is part two of the Details and Stats dashboard, which has a focus on the most common starting and ending letters for names by gender. Names for men and women have certain letter characterisitc that don't appear in one set or the other. One example is how over half of all unique female names end in the letters "a" or "e" while nearly a quarter of all unique male names end in "n". There are also some similarities that transend gender, such as the fact that the most frequent starting letter for both men and women is "A".
